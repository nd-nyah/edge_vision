🧠 Edge Device Offline Video Analysis Platform
Overview
This project is a research-oriented edge AI system for offline video streaming and intelligent video analysis using AI models, agent-based reasoning, and prompt engineering. 
This platform is designed for testing and benchmarking computer vision models, with initial validation using Ultralytics YOLO for real-time object detection on video streams.

🎯 Core Objectives
The platform is designed to:
Process live and offline video streams on edge devices
Run AI-powered video analysis locally without cloud dependency
Support autonomous AI agents for event detection and reasoning
Enable prompt-driven interpretation of visual data
Provide a real-time dashboard for monitoring and analytics
Explore efficient AI deployment on resource-constrained hardware

## Edge Ai Video Analysis Project Structure
- backend/ : FastAPI service
- frontend/ : React dashboard
- models/ : ML models
- videos/ : inputs/ camera
- outputs/ : logs/results
